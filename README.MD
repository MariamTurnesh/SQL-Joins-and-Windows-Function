🔗 SQL Joins
Joins combine data from multiple tables based on related columns.
Types covered:

INNER JOIN – Returns matching rows from both tables
LEFT JOIN – Returns all rows from the left table and matched rows from the right
RIGHT JOIN – Returns all rows from the right table and matched rows from the left
FULL OUTER JOIN – Returns all rows from both tables (includes MySQL workaround)

Each example demonstrates:

Proper join syntax
When to use each join type
Common mistakes and how to avoid them


📊 Window Functions
Window functions perform calculations across rows while keeping individual rows intact.
Categories covered:
Aggregate Functions

SUM(), AVG(), COUNT(), MIN(), MAX()

Ranking Functions

ROW_NUMBER(), RANK(), DENSE_RANK(), PERCENT_RANK()

Value Functions

LAG(), LEAD(), FIRST_VALUE(), LAST_VALUE()

Examples include:

Ranking employees by salary
Calculating running totals
Comparing current vs previous values
Partitioning data by category
